# Genesis

## Building

Requires `nasm` and `ld` on a Linux operating system.

```bash
git clone https://github.com/orosmatthew/hydrogen-cpp
cd genesis-cpp
mkdir build
cmake -S . -B build
cmake --build build
```

Executable will be `gen` in the `build/` directory.
